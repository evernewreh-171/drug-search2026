操作流程

① 匯入 Excel：上傳院內藥品 Excel 檔，系統自動對應欄位並預覽，確認後匯入。

② 管理藥品：可新增、編輯、刪除單筆藥品，即時生效於本機資料。

③ 匯出：下載最新 drugs.json，取代 GitHub 上的 data/drugs.json 即可更新網站。

⚠ 本後台為本機操作，更新網站需手動將 drugs.json 上傳至 GitHub Repository。
